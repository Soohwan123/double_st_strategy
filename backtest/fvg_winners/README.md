### 주요 관찰 (OOS 2년)
- **16/16 모두 수익 + 청산 0건** → robustness 유지
- **5m 이 현실적 수준으로 수렴**: BTC 5m +52~155%, ETH 5m +275~993% (연 환산 수십~수백%)
- **15m 은 여전히 수익 큼**: BTC 15m +1,066~7,664%, ETH 15m +829~54,569% (2년간)
- **MDD 는 전 구간과 대체로 비슷** → 리스크 특성 안정적
- **BTC/ETH 5m ≤40%** 가 가장 균형잡힌 후보:
  - BTC 5m ≤40%: +155% / MDD 32% / 연 환산 ~60%
  - ETH 5m ≤40%: +275% / MDD 37% / 연 환산 ~97%

```bash
cd fvg_winners

# 또는 전체 돌리기
for f in bt_*.py; do venv/bin/python "$f"; done
```

## 버전별 특징
- **v2~v5**: HTF 필터 없음, Single TP
- **v6_htf / v6_1 / v6_2**: HTF 1h EMA200 필터, Single TP
- **v7_partial**: HTF + Partial TP (50% @ RR1, 50% @ RR2)

## 주의사항
- **XRP MDD ≤ 40% 는 "없음"** — XRP 의 2020~2024 대상승으로 인한 drawdown 으로 40% 이하 조합이 존재하지 않음
- 2020~2026 전체 구간의 10⁹% 이상 수익은 6년 복리 + 레버리지 효과 조합 (실전 수익 아님)
- 2024~2026 OOS 결과가 실전 기대치에 더 가까운 지표
- 실제 매매 구현 시 `FVG_LIVE_TRADING_SPEC.md` 필독
- 기간 변경은 `_common.py` 의 `START` / `END` 수정

## 파일 구조
- `_common.py` — 공통 백테스트 로직 (FVG 감지, HTF 필터, Partial TP 포함)
- `bt_01_*.py` ~ `bt_16_*.py` — 개별 설정으로 run
- `FVG_LIVE_TRADING_SPEC.md` — 실제 매매 프로그램 설계서
